HTML Terms

HTML
Hypertext Markup Language,
HTML Element
a component of an HTML document that tells a web browser how to structure and interpret a part of the HTML document.
HTML Tag
a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document
angle bracket
commonly used to enclose a code of some type
HTML Attribute
piece of markup language used to adjust the behavior or display of an HTML element,
HTML comment
These tags tell browsers to ignore anything between them.
HTML object element
represents an external resource
HTML special character
Character entities are a way of encoding special symbols in HTML using a combination of an ampersand (&), a name or a number, and a semicolon (;).
